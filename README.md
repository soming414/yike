# 真诚交友会员管理系统

一个完整的网页端会员管理系统，用于交友会员展示和管理。

## 功能特点

### 前台展示页面
- 卡片式会员展示
- 男女会员分开显示
- 点击卡片查看详细信息
- 响应式设计，支持手机端
- 微信兼容性优化

### 后台管理页面
- 会员信息管理（增删改查）
- 数据统计和图表展示
- 系统设置管理
- 搜索和筛选功能

### 会员信息字段
- 基本信息：姓名、性别、年龄、民族
- 个人特征：身高、体重、生肖、星座
- 背景信息：籍贯、现居地、学历、职业
- 经济状况：月收入
- 交友偏好：是否接受异地恋、自我介绍、喜欢类型

## 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 启动服务器
```bash
npm start
```

### 3. 访问网站
- 前台页面：http://localhost:3000/index.html
- 后台管理：http://localhost:3000/admin.html

## 部署指南

### 本地部署
1. 确保已安装 Node.js (v14+)
2. 克隆或下载项目文件
3. 运行 `npm install`
4. 运行 `npm start`
5. 在浏览器中访问 http://localhost:3000

### 云服务器部署
1. 将项目上传到云服务器
2. 安装 Node.js 和 npm
3. 运行 `npm install --production`
4. 使用 PM2 等进程管理器运行服务器
5. 配置域名和 SSL 证书

### 微信分享优化
- 所有页面已优化微信内嵌浏览器兼容性
- 使用 HTTPS 确保微信能正常访问
- 响应式设计适配各种手机屏幕

## 使用说明

### 前台页面
1. 使用顶部的选项卡切换男女会员或查看全部
2. 点击会员卡片查看详细信息
3. 详细信息包含完整的会员资料
4. 页面底部有安全提示和使用说明

### 后台管理
1. 点击右上角"后台管理"进入管理界面
2. 左侧菜单可切换不同功能模块
3. 会员管理：查看、添加、编辑、删除会员
4. 数据统计：查看各种统计图表
5. 系统设置：配置网站基本信息

### 添加新会员
1. 在后台管理页面点击"添加会员"按钮
2. 填写完整的会员信息
3. 点击"保存会员"完成添加
4. 系统会自动分配ID并显示在列表中

## 技术栈

- 前端：HTML5, CSS3, JavaScript (ES6+)
- 后端：Node.js, Express.js
- 数据存储：本地 JSON 文件（可升级为数据库）
- 图标：Font Awesome
